# Harjutaminee
Versioonihalduse harjutamine GitHubis

See hoidla on loodud versioonihalduse ehk GitHub'i kasutamise harjutamiseks.

Sisu:
- `Harjutamine.txt` — faili esimene versioon sisaldab GitHubi tutvustust.
- Hilisemates kõrvalharudes lisatakse isiklikud märkused ja väike lõik GitHubi ajaloost.

Kasutus:
- Muudatused tehakse eraldi harudes (branches) ja seejärel yhdatakse (merge) põhiharuga (main).
- Selle hoidla eesmärk on harjutada faili loomist, harude tegemist, PR‑ide (pull requests) ja ühendamist.
